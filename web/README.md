# NoWenOS Web Frontend

## Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query
- Zustand
- React Router

## Structure
- `src/api/` shared HTTP client
- `src/app/` app providers and router
- `src/features/` domain modules
- `src/pages/` route pages
- `src/stores/` client state
- `src/types/` shared types
